clinical-ui-boilerplate
=======================

Clinical UI boilerplate for Meteor apps made with Bootstrap-3.


####  Installation  

````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/clinical-ui-boilerplate.git

# And then running it...
sudo mrt
````
